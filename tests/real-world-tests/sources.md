# Test File Sources

This document lists all the open source repositories used to collect test files for comparing rustywind CLI against the Prettier Tailwind CSS plugin.

## Source Repositories

### 1. **Notus Next.js** by Creative Tim
- **Repository**: https://github.com/creativetimofficial/notus-nextjs
- **License**: MIT License
- **Description**: Free and Open Source UI Kit with multiple HTML and Next.js elements. Over 120 frontend components.
- **Files Collected**: 14 files (landing pages, components, navbars, cards, sidebar)
- **Key Features**:
  - Extensive use of Tailwind classes (landing page has 185+ className usages)
  - Complex responsive layouts
  - State modifiers (hover, focus, active)

### 2. **TailAdmin** - Free Next.js Admin Dashboard
- **Repository**: https://github.com/TailAdmin/free-nextjs-admin-dashboard
- **License**: MIT License
- **Description**: Next.js and Tailwind CSS free, open-source admin dashboard template with components and pages for building full-featured back-end dashboards.
- **Files Collected**: 8 files (notification dropdown, modal alerts, calendar, auth forms, dashboard pages)
- **Key Features**:
  - Very high class density (notification dropdown has 92+ className usages)
  - Complex UI components
  - TypeScript + TSX files
  - Modern dashboard patterns

### 3. **Tailwind Next.js Starter Blog** by timlrx
- **Repository**: https://github.com/timlrx/tailwind-nextjs-starter-blog
- **License**: MIT License
- **Description**: Next.js and Tailwind CSS blogging starter template configured with latest technologies for technical writing.
- **Files Collected**: 20 files (layouts, components, theme switcher, navigation)
- **Key Features**:
  - Blog layout components
  - Dark mode support
  - Clean component architecture
  - TypeScript components

### 4. **Play** by TailGrids
- **Repository**: https://github.com/TailGrids/play-tailwind
- **License**: MIT License (based on GitHub info)
- **Description**: Free and open source Tailwind CSS template for Startup, SaaS, Apps, and Business websites.
- **Files Collected**: 4 HTML files (index, blog-details, about, pricing)
- **Key Features**:
  - Pure HTML templates
  - Very high class density (index has 542+ class attributes)
  - Landing page patterns
  - Marketing website layouts

### 5. **Flowbite Vue**
- **Repository**: https://github.com/themesberg/flowbite-vue
- **License**: MIT License
- **Description**: Official Vue 3 components built for Flowbite and Tailwind CSS.
- **Files Collected**: 4 Vue files (pagination, carousel, modal, toast)
- **Key Features**:
  - Vue 3 Single File Components (.vue)
  - Component library patterns
  - Diverse component types
